Like Example 1, but only if you click on the left side of the screen and only logs the clientX and clientY: 
const Rx = require(`rx-dom`);
const mouseDown$ = Rx.DOM.mousedown(document);
mouseDown$
  .map(event => {return {x : event.clientX, y : event.clientY}})
  .filter(e => e.x < 200)
  .subscribe(console.log);
